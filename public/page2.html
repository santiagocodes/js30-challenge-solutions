12-Key Sequence Detection (Konami Code)

This is perfect for a COVID Easter Egg hunt. The origins of coding-in Easter Eggs goes back to the 70s with a disgrunted game designer that was tired of receiving no credit for his work 😁.

Now, Konami Code was created by Kazuhusa Hashimoto in order to give himself power-ups when testing out his game Gradius (which he forgot to remove before launching). That secrete code was up, up, down, down, left, right, left, right, B, A. This is the same code you can now use in pages like Buzzfeed to get, for example, a snowy background.

In the case of my project, when you type in 'santiagocodes' cornify will be activated and a random unicorn and/or rainbow will appear on your screen.

14-Object and Arrays (Reference vs Copy)

14-Object and Arrays (Reference vs Copy)

A good challenge to understand the difference between reference and copy in JavaScript. This is a source of many bugs so, while not elegant, it is important to understand.

The most important thing to take from this challenge is that a reference will always reference back to all the other arrays while a copy won't reference back if you make updates to any of the arrays. The same goes for objects, though the possible solutions are different.

Update: For objects you can now use object spread operator {...obj} instead of using the longer Object.assign({}, obj) solution. For more information, click here.
